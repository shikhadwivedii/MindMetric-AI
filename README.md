# MindMetric AI

MindMetric AI is an AI-powered stress and burnout prediction system built using Machine Learning and Streamlit.

## Features
- Stress level prediction
- Burnout risk analysis
- Wellness recommendations
- Interactive Streamlit dashboard
- Psychological analytics

## Technologies Used
- Python
- Scikit-learn
- Streamlit
- Pandas
- NumPy
- Matplotlib
- Seaborn
